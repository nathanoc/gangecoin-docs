GangeCoin Documentation

Introduction

GangeCoin is a virtual currency which runs entirely on Microsoft Power Automate. Transactions work using commands sent via email. For an email to be registered as a command, it must have the subject line "GNG".

Commands

Note that all commands must have the subject line "GNG".

Balance

Syntax: balance

Shows your current balance in GangeCoin.

Cooldown

Syntax: cooldown [mine, steal]

Shows the length of the cooldown relevant to the argument given.

Create

Syntax: create

Creates an account if you do not have one already.

Give

Syntax: give [target email] [amount]

Sends a given amount of GangeCoin to an address, provided they have an account. The payee will receive an email informing them of the transaction.

If the value paid is negative, a robbery will be attempted. This has a chance of success inversely proportional to the amount attempted to steal, with a minimum probability of 0.1% and a maximum of 50%. If successful, any amount of GangeCoin can be stolen, even if this leaves the victim with a negative balance. If unsuccessful, the thief's balance will be reset to zero, and all their subsequent robbery attempts for a period afterward will also fail.

The target email is case-sensitive.

Help

Syntax: help

Provides a link to the GangeCoin Documentation (this site).

Mine

Syntax: mine

Mines a small amount of GangeCoin.
